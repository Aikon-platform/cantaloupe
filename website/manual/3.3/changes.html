What's New in 3.3

  • Added RESTful API endpoints to purge images from the cache and retrieve and update the configuration.
  • Added an auto-normalize option to all image processors.
  • ImageMagickProcessor supports ImageMagick 7.
  • A cache=false URL query option can be used in all endpoints to bypass the derivative cache per-request.
  • The delegate script is reloaded on change rather than per-request.
