Access Control

Authentication

Cantaloupe has built-in support for HTTP Basic authentication. To enable it, set the following keys in the configuration file:

{% highlight properties %} auth.basic.enabled = true auth.basic.username = myusername auth.basic.secret = mypassword {% endhighlight %}

When enabled, the entire website will be restricted.

HTTP Basic transmits passwords in clear text, so it should only be used on a trusted local network, or in conjunction with TLS.

A reverse proxy server can be used to support other means of authentication.

Authorization

Currently, Cantaloupe is not authorization-aware. But, stay tuned.
