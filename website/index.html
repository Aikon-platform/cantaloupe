About

Cantaloupe is an open-source image server developed at the University of Illinois at Urbana-Champaign Library. By combining the basic operations of cropping, scaling, rotation, flipping, color filtering, and format conversion, it can support on-the-fly derivative image generation and deep zooming into high-resolution source images.

The IIIF Image API is an open standard intended to promote interoperability between clients—like zooming image viewers—and image servers. Cantaloupe supports versions 1.1 and 2.0 of this API. Most of its processors meet the highest compliance level, and should work with any IIIF-compliant client software.

Cantaloupe features a configurable processing pipeline based on:

  • Endpoints, that intercept client requests;
  • Resolvers, that locate and retrieve the source images specified in the requests;
  • Processors, that transform them according to the parameters in an image request; and
  • Caches, that cache generated images.

This makes it highly adaptable, while remaining easy to configure and use.

Features

  • Easy to get working. Cantaloupe is an uncharacteristically simple Java application with no Servlet containers or XML files to deal with.
  • Self-contained for easy prototyping, deployment, and upgrading.
  • Pluggable resolvers for retrieving images from a variety of sources, with scriptable "hooks" to enable dynamic lookup and resolver selection.
  • Pluggable processors to support a variety of source image formats, including JPEG, TIFF, and JPEG2000—as well as still-frame extraction from videos in a variety of formats.
  • Optional server-side caching with configurable aggressiveness.
  • Available built-in authentication.
