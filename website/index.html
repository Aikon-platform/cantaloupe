Cantaloupe is an open-source image server enabling on-demand derivative generation from high-resolution source images. With available processing operations including cropping, scaling, and rotation, it can support deep-zooming image viewers, as well as on-the-fly thumbnail generation. Compliance with the IIIF Image API enables it to work with a growing number of client applications.

Developed at the University of Illinois at Urbana-Champaign Library, Cantaloupe augments an emerging digital image repository containing thousands of high-resolution scanned images. Project goals include performance, scalability, and extensibility, without sacrificing ease-of-use.
